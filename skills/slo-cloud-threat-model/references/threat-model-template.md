## Token rules

- `{{PROVIDER}}` — the scenario `provider` field: `aws`, `github`, `cloudflare`, or `cross-provider`.
- `{{CITATIONS_YAML}}` — one block per deduplicated citation:
  `  - framework: <FW>` / `    id: <ID>` / `    url: <https URL>`. If there are
  zero citations emit `  []`.
- `{{STRIDE_ROWS}}` — one row per `stride[]`. Escape any `|` in text as `\|`;
  collapse newlines to spaces. `Controls` is IDs-only, comma-separated, or
  `(none enumerated)`.
- `{{CITATION_ROWS}}` — one row per deduplicated `{framework, id, url}`, first-seen order.
- `{{COMPONENT_BULLETS}}` — `` - `<name>` `` — `<availability>`. `<rationale>`.
  Preserve the pinned availability verbatim.
- `{{OPEN_QUESTIONS}}` — one bullet per required framework with zero citations and
  per control ID with an unknown framework; otherwise a single bullet stating no
  unresolved questions were recorded at generation time.
