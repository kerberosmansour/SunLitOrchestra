# Variation template — CWE-416 Use After Free

## Why this class

The borrow checker prevents UAF in safe Rust. UAF surfaces in:
- `unsafe { }` blocks where raw pointers are dereferenced after the owner has dropped.
- FFI bindings where a C API holds a callback pointer outliving the Rust value it captured.
- Concurrency primitives where a token / handle outlives its owner across threads.

## Sink shapes (3 minimum)

### 1. `raw_pointer_deref_after_drop`

```rust
unsafe fn use_after_drop() {
    let v = vec![1, 2, 3];
    let p = v.as_ptr();
    drop(v);
    let _ = *p; // UAF
}
```

### 2. `vec_set_len_then_use_old_ptr`

```rust
unsafe fn vec_realloc_use_after() {
    let mut v: Vec<u8> = Vec::with_capacity(4);
    let p = v.as_mut_ptr();
    v.set_len(8);
    v.push(1); // realloc; p is now dangling
    *p = 9; // UAF
}
```

### 3. `ffi_callback_outliving_owner`

```rust
extern "C" fn cb(ctx: *mut std::ffi::c_void) {
    unsafe {
        let s = &*(ctx as *const String);
        println!("{s}");
    }
}

fn register() {
    let s = String::from("hello");
    unsafe {
        ffi_register_callback(cb, &s as *const _ as *mut _);
    }
    // s drops here; cb fires later via FFI; UAF
}
extern "C" { fn ffi_register_callback(cb: extern "C" fn(*mut std::ffi::c_void), ctx: *mut std::ffi::c_void); }
```

## Pattern shape strategy

Until the M1 smoke-test confirms `pattern-inside: unsafe { ... }` works directly, use the workaround:

```yaml
pattern-inside: |
  fn $F(...) { ... unsafe { ... } ... }
```

Or scope by raw-pointer type:

```yaml
pattern-either:
  - patterns:
      - pattern: unsafe { ... *$P ... }
      - pattern-inside: |
          fn $F(...) { ... drop($X); ... }
```

Cross-procedural UAF detection is genuinely beyond Semgrep CE's reach for v1; this rule catches the local syntactic shapes.

## Good (silent) shapes for the fixture

```rust
fn safe_owned() {
    let v = vec![1, 2, 3];
    let _sum: i32 = v.iter().sum();
} // v drops here; nothing dangling

fn safe_arc_share(arc: std::sync::Arc<String>) {
    let cloned = arc.clone();
    std::thread::spawn(move || {
        println!("{cloned}");
    });
}
```
